Filamentos
  
  Inventario web para gestionar tus carretes de filamento de impresión 3D, con soporte nativo para el catálogo de Bambu Lab.

  Características

  - 📦 Gestión de spools: registra carretes con código de 5 dígitos de Bambu Lab y obtén automáticamente familia, color, temperaturas y acabado.
  - 🎨 Swatches reales: visualización por color con forma de carrete y porcentaje restante.
  - 🗂️  Catálogo completo: navega líneas PLA Basic, Matte, CF, PETG, ABS, TPU y más, con conteo de spools por subcategoría.
  - 🎰 Filtro por slot AMS: agrupa y filtra por slot del Bambu AMS para planificar impresiones multicolor.
  - 📊 Ordenamiento y vistas plegables: organiza spools por color, familia, uso o restante.
  - 🗄️  Archivar/desarchivar: oculta spools agotados sin perder el historial.
  - ⚡ 100% estático: corre como SPA en GitHub Pages, sin backend ni build step.

  Stack

  HTML + React (UMD) + Babel standalone + CSS vanilla. Todo el catálogo de Bambu Lab vive embebido en index.html.

  Uso
  
  Abre index.html localmente o visita la versión publicada en GitHub Pages.
